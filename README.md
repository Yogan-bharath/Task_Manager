# 📝 Task Manager

A responsive **Task Manager** web application built using **HTML, CSS, and Vanilla JavaScript**. This project demonstrates complete **CRUD (Create, Read, Update, Delete)** functionality using DOM manipulation, allowing users to efficiently manage their daily tasks through a clean and interactive interface.

## 🌐 Live Demo

🔗 **https://yogan-bharath.github.io/Task_Manager/**

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 📋 View all tasks in an organized layout
- 🔍 Filter tasks (All, Active, Completed)
- 📊 Real-time task statistics
- ⚡ Dynamic DOM updates without page refresh
- 📱 Fully responsive design
- 💾 Store tasks using Local Storage
- 🌙  Theme Toggle
- 🏷️ Task Categories
- 🔍 Search Functionality
